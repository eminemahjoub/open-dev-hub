# Releases

## v1.0.0 - Initial Release (November 5, 2025)

Welcome to the first release of **open-dev-hub** — a curated hub of open-source resources for developers!

### What's New
- Complete repository structure with categories for AI/ML, Web Development, Cybersecurity, Dev Tools, and IoT.
- Professional README with badges, logo, and contribution guidelines.
- GitHub Actions CI for linting and quality checks.
- Starter project templates for quick prototyping.
- Comprehensive contributing docs and code of conduct.

### Helpful Open-Source Links
Here are some recommended open-source projects and resources to get you started:

- **AI/ML**: [Hugging Face Transformers](https://github.com/huggingface/transformers) — State-of-the-art NLP models.
- **Web Development**: [Next.js](https://github.com/vercel/next.js) — React framework for production.
- **Cybersecurity**: [OWASP ZAP](https://github.com/zaproxy/zaproxy) — Web app security scanner.
- **Dev Tools**: [VS Code](https://github.com/microsoft/vscode) — Popular code editor.
- **IoT**: [Home Assistant](https://github.com/home-assistant/core) — Open-source home automation.

### Installation
Clone the repo and explore the categories:
```bash
git clone https://github.com/eminemahjoub/open-dev-hub.git
cd open-dev-hub
```
